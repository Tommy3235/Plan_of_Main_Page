# Plan of Main Page (PMP) 主页计划
## 简介
各种导航页面很多，没有一个是有良心的。  
我最终做出了一个决定，那就是成为偶像（大雾）。  
我决定自己写一个属于自己的导航页面，这是一个良心的个人导航页，收录了一些我常用的网址。  
刚刚开始这个项目，还有很大的进步空间。
